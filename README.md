# Simple Audio App

## Overview
The Simple Audio App is an Android application that allows users to control audio settings, including volume, bass enhancement, and volume boost features. This project aims to provide a user-friendly interface for audio manipulation on mobile devices.

## Features
- **Volume Control**: Adjust the overall volume of the audio output.
- **Bass Control**: Enhance the bass levels for a richer audio experience.
- **Volume Boost**: Temporarily increase the volume above the maximum level.

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/thethethe78/simple-audio-app.git
   ```
2. Open the project in Android Studio.
3. Build and run the application on an Android device.

## Usage
- Launch the app from your device.
- Use the sliders to adjust volume and bass levels.
- Enable volume boost for higher audio output.
